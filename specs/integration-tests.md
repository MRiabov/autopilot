# BMAD Autopilot Integration Tests

This catalog is owned by the standalone `.autopilot` product repository.
It is intentionally separate from the main repo's `specs/integration-tests.md`.

## P1 Autopilot coverage

| ID      | Test                                                         | Required assertions                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| ------- | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| INT-206 | BMAD autopilot dirty-worktree confirmation gate              | BMAD Autopilot must require explicit `y/yes` confirmation before proceeding when the active workspace is dirty, and must abort without side effects when the confirmation is rejected or stdin is unavailable.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| INT-207 | BMAD autopilot empty-review fail-closed gate                 | BMAD Autopilot code review must fail closed when the current workspace snapshot has no reviewable source, must not continue to a passing review on an empty scope, must not call the review tool in that case, and must reroute back to development instead of hard-blocking the run.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| INT-208 | BMAD autopilot workspace-scoped review artifacts             | BMAD Autopilot review artifacts must persist under the active workspace root and be discoverable from that same workspace root instead of a stale repository-root review directory.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| INT-209 | BMAD autopilot QA fail reroute gate                          | BMAD Autopilot must reroute a failed story QA pass back to story development instead of hard-blocking the run, and the story file plus sprint-status entry must move back to `in-progress`.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| INT-210 | BMAD autopilot dev validation reroute gate                   | BMAD Autopilot must reroute invalid story-development output back to story development instead of hard-blocking the run, and the story file plus sprint-status entry must move back to `in-progress`.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| INT-211 | BMAD autopilot code-review validation reroute gate           | BMAD Autopilot must reroute invalid story code-review output back to story development instead of hard-blocking the run, and the story file plus sprint-status entry must move back to `in-progress`.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| INT-212 | BMAD autopilot transient dev-block reroute gate              | BMAD Autopilot must treat a transient `stories_blocked` dev output as a retryable signal, keep the story in `in-progress`, reroute back to development immediately, and never persist a terminal blocked state for that case.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| INT-213 | BMAD autopilot quota-exhaustion retry gate                   | When Codex reports quota exhaustion, BMAD Autopilot must keep running, switch to a healthier account when one exists, or wait and retry when no alternative account is available, without terminating the workflow.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| INT-214 | BMAD autopilot dirty-worktree bypass flag gate               | `--accept-dirty-worktree` must bypass the interactive dirty-tree prompt, continue without waiting for stdin, and still proceed through the active workflow.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
